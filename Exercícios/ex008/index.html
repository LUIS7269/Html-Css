Principais Formatações

Negrito/Destaque

Podemos usar o negrito tanto com a tag B quanto com a tag STRONG que elas não iram mudar nada visualmente o que muda, é que a tag STRONG diferente da tag b tem valor semântico(significado)

Itálico/Ênfase

Neste caso é a mesma coisa, podemos usar a tag I para acrescentar o itálico quanto a tag EM para destacar a frase o que difere as duas é que a tag EM possui valor semântico(significado)

Marcação de texto

Podemos criar um texto marcado usando a tag MARK

E neste de baixo também temos um texto marcado

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno , usando a tag BIG e a tag SMALL, POREM a tag big "morreu" e pode sair de funcionamento a qualquer momento.

Texto deletado

Podemos marcar um texto como deletado para mostrar que ele deve ser lido mas, não considerado. Usando a tag DEL

Texto inserido

Podemos marcar um texto como inserido para, mostrar que ele foi adicionado posteriormente. Usando a tag INS

Para isso existe também a TAG U (não semântica)

Texto sobrescrito

Usando por exemplo em: x20+3. Utilizando a tag SUP

Texto subscrito

Usado por exemplo em : H2O. Utilizando a tag SUB
